# Invitación para Alisson

## Vista previa local
Desde esta carpeta ejecuta `python -m http.server 8000` y abre http://localhost:8000.

## Personalización
- Cambia el número en `script.js`, en `const WHATSAPP_NUMBER`.
- Edita los textos visibles directamente en `index.html`.
- Para agregar música autorizada, reemplaza el efecto Web Audio de `playChime()` en `script.js` por un archivo local con control manual; no se reproduce automáticamente.

## Publicación y retiro
Sube la carpeta a un repositorio de GitHub, activa GitHub Pages desde Settings → Pages y usa la rama principal. Para retirar la página, desactiva Pages o elimina el repositorio.
